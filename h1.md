# H1 Scope First, Controls Later

Tehtävänä on kuvata oma kotiverkko, jossa kurssitehtäviä suoritetaan. Rajataan kotiverkosta se
alue, johon ISMS (Information Security Management System = Tietoturvallisuuden hallintajärjestelmä)
koskee sekä tunnistetaan rajapinnat.

## a) Perustaso
### 1. Mitä kuuluu soveltamisalaan
#### Kotiverkon perusinfra
- 5G Reititin
- Wi-Fi
#### Kurssiharjoituksiin käytettävät laitteet:
- Kannettava tietokone
- Puhelin (MFA)
- Linux virtuaalikone VirtualBoxissa
#### Tiedot:
- Kurssimateriaalit:
  - Moodle
  - Omat muistiinpanot (OneDrive)
  - Kurssi Tero Karvisen nettisivuilla: https://terokarvinen.com/application-hacking/
  - GitHub (kurssitehtävien raportointi)
- Tunnukset ja salasanat: Kotiverkko, oma tietokone, Haaga-Helia, virtuualikone
### 2. Mitä rajaat ulos ja miksi?
- Poikaystävän pöytäkone: Ei ole minun hallinnassa
- Pelikonsoli: En käytä kurssin suoritukseen
- TV: En käytä kurssin suoritukseen
- E-lukulaite: En käytä kurssin suoritukseen
- Elisan verkko: Ei ole minun hallinnassa
### 3. Keskeiset rajapinnat
#### Pilvipalvelut:
- GitHub
- Haaga-Helia OneDrive
#### Koti- ja ulkoverkon raja
- Reititin ja palomuuri kotiverkon ja internetin välissä
#### Toimittajat
- ISP: Elisa
- Laitteet:
  - Reititin: ZTE
  - Kannettava tietokone: Lenovo
  - Puhelin: Samsung
- Pilvipalvelut: GitHub ja Microsoft
### Mitä näyttöä voisin esittää?
-	Reitittimen ja palomuurin asetukset
-	Kannettavan tietokoneen tiedot (esim. IP-osoite)
-	Puhelimen Authenticator -sovellus
-	Virtual Box: Listaus virtuaalikoneista
-	OneDrive varmuuskopiointiasetukset
-	Lista kotiverkossa olevista laitteista
